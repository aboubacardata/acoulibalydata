Understanding the Split

Using the Box-Cox transformation and 1.5 standard deviation threshold, we identified a clear separation point at $98.42 daily average NMV. This means:

  • Normal Users (93.39%): Spend less than $98.42/day on average - typical individual consumers
  • Resellers (6.61%): Spend $98.42/day or more - commercial buyers purchasing for resale

This segmentation enables targeted strategies: personalized support for high-value resellers while optimizing the experience for the majority of normal users.

Core Algorithm (Python) 
# Apply Box-Cox Transformation
data_transformed, best_lambda = stats.boxcox(df['NMV_Shifted'])
mean_trans = np.mean(data_transformed)
std_trans = np.std(data_transformed)

# Calculate threshold (1.5 SD above mean)
limit_trans = mean_trans + (1.5 * std_trans)

# Inverse transform to original scale
def inv_boxcox(y, ld, shift):
    if ld == 0: 
        return np.exp(y) - shift
    else: 
        return (y * ld + 1)**(1/ld) - shift

threshold_original = inv_boxcox(limit_trans, best_lambda, shift_value)

# Classify users
df['is_reseller'] = df['DAY_AVG_NMV'] > threshold_original
